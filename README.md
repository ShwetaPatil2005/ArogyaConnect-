# 🏥 ArogyaConnect - Appointment Booking System

## 📌 Project Description
ArogyaConnect is a healthcare appointment booking system that allows patients to easily schedule appointments with doctors while managing time slots efficiently.

## 🚀 Key Features
- 👤 User Registration & Login
- 🩺 Doctor Availability Management
- 📅 Appointment Booking System
- ⏰ Time Slot & Break Handling
- 💾 MySQL Database Integration
- 🔐 Secure Authentication System

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: (Add yours)
- Database: MySQL

## 📂 Project Structure
- `/frontend` → UI pages
- `/backend` → Server logic
- `/database` → SQL files

## ⚙️ How to Run
1. Clone the repository
2. Import SQL file into MySQL
3. Start backend server
4. Run frontend
